Music:

In Audacity exporting your music file as a 16-bit WAV is the reccomended type. (48000hz)

1) Open Rad Video Tools

2) Bink the .wav file you created, name it whatever.

3) Once the conversion is completed. Change the .bik file name to the specific monster you'd like to chance. (examples below)

4) Replace the file in DATA/files/video.

Example of monster music file names:

Alien_01_Gigan_Theme_Rev.bik

Earth_Defenders_04_Baragon.bik

Mecha_04_Jet_Jaguar.bik
